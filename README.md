# shiro-freemarker


### 说明

 > 基于[shiro-freemarker-tags](https://github.com/zhoushuaichang/shiro-freemarker-tags) 添加一些新的逻辑


### Maven

``` xml
<dependency>
	<groupId>${project.groupId}</groupId>
	<artifactId>shiro-freemarker</artifactId>
	<version>${project.version}</version>
</dependency>
```
